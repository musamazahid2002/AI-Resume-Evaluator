# AI-Resume-Evaluator
AI Resume Evaluator is an intelligent web-based application that helps job seekers improve their resumes using Artificial Intelligence. The system allows users to upload their resumes in PDF or DOCX format and automatically analyzes the content, structure, skills and overall quality of the resume.
